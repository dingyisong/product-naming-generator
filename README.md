# 🏷️ 产品功能命名生成器

> 输入功能描述，一键生成 10 个多维度命名方案并智能评分

**在线体验：** https://dingyisong.github.io/product-naming-generator/

---

## ✨ 功能特性

- 📝 **输入三要素**：功能描述 + 品牌调性（可选）+ 目标用户（可选）
- 💡 **生成10个命名方案**，每个附带详细命名理由
- 🎯 **四大命名维度**：直白型 / 隐喻型 / 情感型 / 动词型
- 📊 **多维度评分**：易发音、易记忆、相关性、品牌感（0-100分）
- ✅ **可行性评估**：发音难度、记忆难度、商标注册风险
- 🏆 **综合排名**：按加权分自动排序，金银铜牌标注
- 🔍 **类型筛选**：快速按命名维度过滤
- 📋 **一键复制**：点击名称卡片直接复制

## 📋 命名评估框架

| 维度 | 说明 |
|------|------|
| 🗣️ 易发音 | 音节数（2-4最佳）、有无绕口组合、朗朗上口程度 |
| 🧠 易记忆 | 与已知词汇关联度、视觉辨识度、记忆钩子 |
| 🎯 相关性 | 与功能的语义契合度，看名识功能的难易程度 |
| ✨ 品牌契合 | 与品牌调性一致性、视觉延展性、家族命名融合度 |
| ©️ 商标风险 | 通用词汇占比、现有同名产品、造词独特性 |
| 🌍 国际化 | 英文/拼音难度、跨文化歧义风险、海外扩展潜力 |
